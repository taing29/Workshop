### Week 6 Objectives:

* Learn how to build scalable and highly available systems using AWS Auto Scaling.
* Practice monitoring AWS resources and application performance with Amazon CloudWatch.
* Understand how Auto Scaling and CloudWatch work together to improve application availability and reliability.

### Week 6 Achievements:

* Learned how to design and deploy an auto-scaling infrastructure using Launch Templates, Elastic Load Balancing, and Auto Scaling Groups to improve system availability.
* Gained hands-on experience in testing automatic scaling by simulating workloads and verifying that the system could dynamically adjust its resources based on demand.
* Developed practical skills in using Amazon CloudWatch to monitor system performance through Metrics, Logs, Alarms, and Dashboards.
* Understood how Amazon CloudWatch integrates with Auto Scaling to automatically respond to resource utilization changes, enabling more reliable and efficient application management.
