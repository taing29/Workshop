### Week 7 Objectives:

* Learn how to manage AWS resources using the AWS Command Line Interface (AWS CLI).
* Practice interacting with various AWS services through AWS CLI.
* Become familiar with Amazon DynamoDB and accessing NoSQL databases using Python.

### Week 7 Achievements:

* Learned how to install, configure, and use AWS CLI to efficiently manage AWS resources through the command-line interface.
* Gained hands-on experience using AWS CLI to perform operations on Amazon S3, Amazon SNS, IAM, Amazon VPC, and to launch and manage Amazon EC2 instances.
* Understood the AWS CLI authentication process, including configuring credentials and securely accessing AWS services.
* Became familiar with Amazon DynamoDB and successfully used Python to connect to the database and perform basic data operations such as reading and writing data in a NoSQL environment.
