Throughout **11 weeks** of participating in the **First Cloud AI Journey** program, I learned and gained hands-on experience with core Amazon Web Services (AWS) services, ranging from infrastructure management, networking, and security to application deployment and cloud monitoring.

**Week 1:** [Getting started with AWS, creating an AWS Free Tier account, and exploring core AWS services.](1.1-week1/)

**Week 2:** [Managing costs with AWS Budgets, configuring IAM, and deploying networking infrastructure using Amazon VPC.](1.2-week2/)

**Week 3:** [Working with Amazon EC2, deploying applications on Windows and Linux, and managing access with IAM Roles.](1.3-week3/)

**Week 4:** [Hosting static websites with Amazon S3 and CloudFront, and building applications using Amazon RDS.](1.4-week4/)

**Week 5:** [Deploying applications with Amazon Lightsail and Amazon Lightsail Containers.](1.5-week5/)

**Week 6:** [Building scalable applications with EC2 Auto Scaling and monitoring resources using Amazon CloudWatch.](1.6-week6/)

**Week 7:** [Managing AWS resources with AWS CLI and working with Amazon DynamoDB.](1.7-week7/)

**Week 8:** [Exploring Amazon ElastiCache, Amazon CloudFront, and application performance optimization.](1.8-week8/)

**Week 9:** [Configuring Amazon CloudFront with Lambda@Edge and deploying Amazon WorkSpaces.](1.9-week9/)

**Week 10:** [Working with AWS Managed Microsoft AD, AWS Lambda, and monitoring infrastructure using Amazon CloudWatch and Grafana.](1.10-week10/)

**Week 11:** [Designing the ReviewSentinel system architecture, security model, data model, and preparing the AWS infrastructure for deployment.](1.11-week11/)

**Week 12:** [Deploying the complete ReviewSentinel serverless application on AWS, integrating backend and frontend services, and performing end-to-end system testing.](1.12-week12/)
