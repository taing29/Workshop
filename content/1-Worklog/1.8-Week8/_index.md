### Week 8 Objectives:

* Learn about Amazon ElastiCache and how in-memory caching improves application performance.
* Practice using the AWS SDK to connect to and manage data in Amazon ElastiCache.
* Learn how to host static websites on Amazon S3 and deliver content through Amazon CloudFront.

### Week 8 Achievements:

* Gained an understanding of how Amazon ElastiCache improves application performance by storing frequently accessed data in memory, reducing database workload and improving response times.
* Successfully created an Amazon ElastiCache Cluster and used the AWS SDK to connect to the cluster, performing basic read, write, and data management operations.
* Learned how to host a static website on Amazon S3 by creating a bucket, uploading website files, and configuring the required settings for public access.
* Configured Amazon CloudFront to distribute content from Amazon S3 and understood how a Content Delivery Network (CDN) improves website performance by reducing latency and accelerating content delivery.
* Developed a practical understanding of integrating Amazon S3, Amazon CloudFront, and Amazon ElastiCache to build a scalable, high-performance architecture for web applications on AWS.
