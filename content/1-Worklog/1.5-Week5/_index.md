### Week 5 Objectives:

* Learn how to deploy applications and databases using Amazon Lightsail.
* Practice managing, securing, and monitoring resources on Amazon Lightsail.
* Become familiar with Amazon Lightsail Containers and the process of deploying containerized applications.

### Week 5 Achievements:

* Learned how to deploy applications and databases on Amazon Lightsail, including WordPress, PrestaShop, and managed database instances.
* Gained hands-on experience with Lightsail management features such as creating Snapshots, upgrading instances, configuring Alarms, and applying security best practices.
* Successfully created and managed Amazon Lightsail Containers, and deployed applications using both Public Images and custom Container Images.
* Developed an understanding of how Amazon Lightsail provides a simple and cost-effective platform for deploying and managing web applications and services.
