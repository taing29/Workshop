### Week 3 Objectives:

* Learn and practice deploying Amazon EC2 instances on both Windows and Linux.
* Understand how to deploy applications on EC2 and manage access permissions using AWS IAM.
* Become familiar with Access Keys and IAM Roles for secure access to AWS resources.

### Week 3 Achievements:

* Learned the process of launching, configuring, and managing Amazon EC2 instances on both Windows and Amazon Linux.
* Successfully deployed applications on EC2, including a user management application and a Node.js application running on Amazon EC2 Windows.
* Gained hands-on experience with Access Keys, IAM Roles, and Instance Profiles to securely grant permissions to EC2 instances.
