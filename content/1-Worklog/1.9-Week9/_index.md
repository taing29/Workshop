### Week 9 Objectives:

* Learn how to configure and optimize Amazon CloudFront with Lambda@Edge.
* Practice implementing advanced CloudFront features to improve website performance and security.
* Become familiar with Amazon WorkSpaces and the deployment process for virtual desktop environments on AWS.

### Week 9 Achievements:

* Learned how to configure Amazon CloudFront by creating distributions, configuring cache behaviors, response headers, origin groups, and custom error pages to optimize content delivery.
* Successfully deployed and integrated a Lambda@Edge function with Amazon CloudFront to process requests at edge locations, improving application performance and enabling advanced request handling.
* Gained practical experience monitoring CloudFront Metrics and Logs to evaluate performance, troubleshoot issues, and ensure reliable content delivery.
* Built the required AWS infrastructure for Amazon WorkSpaces by configuring a VPC, Subnets, Internet Gateway, NAT Gateway, and Amazon EC2 instances.
* Learned how to deploy, access, and manage Amazon WorkSpaces through both a web browser and the WorkSpaces Client application, while gaining an understanding of the infrastructure required to support virtual desktop environments on AWS.
