### Week 2 Objectives:

* Learn about AWS services for cost management, identity and access management, and networking.
* Practice configuring AWS Budgets, AWS Identity and Access Management (IAM), and Amazon Virtual Private Cloud (VPC).
* Understand how to securely deploy and manage AWS resources following best practices.

### Week 2 Achievements:

* Learned how to use AWS Budgets to monitor cloud costs and configure different types of budgets, including Cost, Usage, RI, and Savings Plans Budgets.
* Gained a solid understanding of AWS IAM by creating users, groups, and roles, as well as managing permissions through role switching.
* Understood the fundamentals of Amazon VPC, including firewall configuration, EC2 deployment, and Site-to-Site VPN connectivity.
* Became familiar with Infrastructure as Code (IaC) concepts and the basic process of provisioning and managing AWS infrastructure through templates.
