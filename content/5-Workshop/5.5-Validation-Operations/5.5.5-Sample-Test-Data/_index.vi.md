# Dữ liệu test mẫu

## Tổng quan

Dùng dữ liệu review mẫu trong hướng dẫn để kiểm tra toàn bộ pipeline.

### Những gì cần test

- Đăng ký và đăng nhập Cognito
- Upload review qua trình duyệt
- Processor Lambda chạy đúng
- Analyzer Lambda chạy đúng
- Dashboard cập nhật dữ liệu

### Ghi chú

1. Giữ file mẫu thật nhỏ để dễ upload và theo dõi.
2. Dùng đúng format mẫu trong hướng dẫn triển khai.

### Kết quả mong đợi

Dữ liệu mẫu phải đi qua toàn bộ pipeline mà không cần can thiệp thủ công.
